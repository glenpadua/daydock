# Sample Vault

This folder demonstrates a minimal Obsidian-compatible vault layout for DayDock.

## Structure

```text
sample-vault/
  Daily Notes/
    2026-03-11.md
```

## How to use

1. Open DayDock settings.
2. Set `Vault folder` to this `sample-vault` directory.
3. Keep:
   - `Daily notes folder`: `Daily Notes`
   - `Filename format`: `YYYY-MM-DD`
   - `ifTimeSectionPattern`: `if time`

## Notes

- `Evening Close` is plain markdown and renders as a normal section.
- Time-blocked tasks follow `HH:MM - HH:MM → Task`.
- Non-time-blocked tasks are fully supported.
